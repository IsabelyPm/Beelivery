# 🐝 Beelivery - Delivery de bebidas

 **Beelivery** é uma plataforma de e-commerce otimizada para o delivery de bebidas. O projeto foca em uma interface limpa, rápida e extremamente intuitiva para o consumidor final.

## 🛠️ Tecnologias Utilizadas
* **React** + **Vite** (Frontend de alta performance)
* **Tailwind CSS** (Estilização moderna e responsiva)
* **Lucide React** (Ícones minimalistas)
* **Framer Motion** (Animações fluidas)
* **TypeScript** (Segurança e tipagem de dados)

## ✨ Funcionalidades
- [x] Catálogo de produtos dinâmico dividido por categorias.
- [x] Sistema de busca em tempo real.
- [x] Carrinho de compras interativo.
- [x] Interface totalmente adaptável para dispositivos móveis.
- [x] Checkout simulado para validação de fluxo de compra.

## 🚀 Como rodar o projeto localmente
1. Clone o repositório: `git clone https://github.com/seu-usuario/beelivery-buzz.git`
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm run dev`
